PC Build Recommendation System
A web-based system that helps users build a compatible and budget-friendly PC based on their needs such as gaming, programming, video editing, or general use.
🚀 Features

Recommend PC components based on budget
Suggest compatible hardware
Display total build cost
Admin panel for managing components
🛠 Technologies Used
Frontend

HTML


CSS


Backend


Python (Flask)
